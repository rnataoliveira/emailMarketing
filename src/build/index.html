DESENVOLVA EM TODAS AS PLATAFORMAS
Aprenda uma maneira de criar aplicativos com Angular e reutilize seu código e habilidades para criar aplicativos para qualquer destino de implantação. Para web, web mobile, celular nativo e desktop nativo.
VELOCIDADE & DESEMPENHO
Obtenha a máxima velocidade possível na plataforma da Web hoje, e adquira ainda mais, via Web Workers e renderização do lado do servidor. Angular coloca você no controle sobre escalabilidade. Conheça requisitos de dados enormes ao construir modelos de dados em RxJS, Immutable.js ou outro modelo de push.
FERRAMENTAS INCRÍVEIS
Crie recursos rapidamente com modelos simples e declarativos. Estenda o idioma do modelo com seus próprios componentes e use uma ampla gama de componentes existentes. Obtenha ajuda e feedback específicos angulares imediatos com quase todos os IDE e editor. Tudo isso se junta para que você possa se concentrar em criar aplicativos incríveis ao invés de tentar fazer o código funcionar
